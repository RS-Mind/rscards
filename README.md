# rscards
Cards for ROUNDS focused on adding variety with vanilla stats

## Current Card List:

Converts bounces to raw power
- +50% DMG
- +50% Bullet speed
- -2 Bullet bounces

### Changeup

Bullets keep the default trajectory
- +30% ATKSPD
- -25% Projectile Speed

### Hitscan

(Consider disabling when playing with more mods)
- +10000% Bullet speed
- No Bullet bounces

### Open Chamber

- -0.5s Reload time
- -25% Bullet Speed

- +150% ATKSPD

### Slug
